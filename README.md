#PERSONAL FINANCE TRACKER

Personal Finance Tracker is basically a program which is made up of Python, Streamlit and Pandas. It has simple, easy to use and interactive UI, users can enter their expenses with some details like date and category and the program will save it to track the expense. User can see their expense list with the details(date and category) anytime by just clicking on the button "List of expenses" and a new table will be visible containing the details.

FEATURES
1. Easy to use
2. Tracks the expenses with details
3. Stores expenses for detailed information

HOW IT WORKS

The user enter the Date, amount and select category of expense

When user click the "Done" button, the information will be saved

If user wants to see the list of expenses then they would click on "List of expenses" button

A table will be visible with the information like date, category and amount in a sequence

TECH STACK

Python programming with it's data structures like "List" and "Dictionary"

File handling with Python
